# Vuejs 


## Setup

1. Make sure you have Node JS installed. 

```

```

2. Clone the repo:

```
git clone https://github.com/annamontecalvo/vuejs.git
```

3. Open the project folder:

```
cd vuejs
```

4. Install packages and dependencies:

```
npm install
```

5. Start a local dev server at `http://localhost:8080`:

```
npm run serve
```
